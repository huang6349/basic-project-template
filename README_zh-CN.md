Install
-------
[安装依赖]

```bash
node >= 10.13.0
```

```bash
$ npm install -g yarn tyarn
$ tyarn -v
$ tyarn install
```

Usage
-------
[用法]

#### development[开发环境]

```bash
$ yarn run start # with mock
```

```bash
$ yarn run start:no-mock # no mock
```

访问：[http://127.0.0.1:8000](http://127.0.0.1:8000)

#### production[生产环境]

```bash
$ yarn run build # build
```

```bash
$ yarn run analyze # analyze
```
