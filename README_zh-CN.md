安装依赖
-------

```bash
node >= 10.13.0
```

```bash
$ npm install -g yarn tyarn
$ tyarn -v
$ tyarn install
```

使用方法
-------

#### 开发环境

```bash
$ yarn run start
```

```bash
$ yarn run start:no-mock
```

访问：[http://127.0.0.1:8000](http://127.0.0.1:8000)

#### 生产环境

```bash
$ yarn run build
```

```bash
$ yarn run analyze
```
